# 医学影像智能分析系统（MedicalImager）

### 项目简介
面向放射科医师的桌面端 AI 辅助阅片工具，支持 NIfTI 格式三维医学影像的本地化管理、多平面重建（MPR）浏览、窗宽窗位调节以及基于 nnUNet 等自训练模型的器官自动分割。一站式覆盖「影像加载 → 三平面可视化 → AI 分割 → 结果回显」全流程。
